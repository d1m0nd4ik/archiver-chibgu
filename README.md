# VK Archiver CHIBGU

Десктопное приложение для скачивания, обработки, индексации и анализа контента из сообществ ВКонтакте. Разработано для ведения архива публикаций с привязкой к преподавателям и кафедрам ЧИБГУ.

Интерфейс построен на PySide6 (Qt). Данные хранятся в SQLite (`Archive.db`), медиафайлы — в каталоге `Exports_data` с разбивкой по дате публикации.

---

## Основные возможности

### Загрузка и сохранение медиа

- Скачивание постов из группы ВК по токену API (фото, видео, клипы).
- **Ручное добавление** на вкладке «Загрузка контента»: фото, видео и текст без ВК; подпись источника (список + свой вариант), ручные теги.
- **Планировщик** (Настройки): догрузка из ВК и обновление метрик **пока приложение открыто** — не требует работы 24/7.
- **Резервная копия**: ZIP с `Archive.db` и `Exports_data` из настроек.
- Проверка **дубликатов** перед ручным добавлением.
- Фотографии в **JPEG** без перекодирования; видео — без изменения кодека (yt-dlp при необходимости).

### Индексация и поиск

- Полнотекстовый поиск FTS5 по **тексту, тегам и подписи источника** (`source_label`).
- Страница **«Поиск в архиве»**: фильтры (дата, тег, кафедра, автор, медиа, источник), сортировка, **пагинация** (по 100, счётчик «показано N–M из K»).
- Колонки: превью фото, текст, теги, лайки, комментарии, репосты.
- **Пресеты фильтров** (JSON `search_presets.json`).
- Кнопка **«Открыть в хранилище»** — прокрутка к карточке поста.
- **Проверка целостности** (Настройки → Архив): удалить битые записи БД, привязать лишний файл, открыть папку поста.

### Хранилище и сводка

- **Сводка архива** (главный экран): всего постов, за месяц, топ-кафедра, без автора / тегов / вложений.
- Карточки постов с медиа, **«Редактировать»**, временная папка на рабочий стол.
- Кэш миниатюр в `.cache/thumbs`.

### Управление и теги

- **Управление постами**: массовое удаление, смена источника для ручных постов; пересчёт тегов — на вкладке «Тэги».
- **Тэги**: словарь с **иерархией** (родительский хэштег), пересчёт архива.
- **Фоновые задачи** (Настройки → вкладка): загрузка, retag, синхронизация кафедр, целостность — одна очередь с журналом.

### Преподаватели и кафедры

- Справочник, синхронизация с сайта ЧИБГУ, отчёт по кафедре за период (Word) — `core/department_report.py`.
- «Преподаватели в постах», экспорт Word / CSV / Excel.

### Оформление

- Светлая / тёмная тема; выпадающие списки со стрелкой.
- Журналы: `logs/vk_archiver.log`, `logs/errors.log` (ротация); просмотр ошибок — Настройки → Журнал.

## Требования и установка

- Python 3.9+
- `pip install -r requirements.txt`
- `.env` — токен VK, ссылка на группу, тема (см. `.env.example`)

```bash
python main.py
```

### Тесты

```bash
python -m pytest tests/ -q
# или
python -m unittest tests.test_core
```

## Планировщик

Включите в **Настройки → Основные**. Интервал: ежедневно или еженедельно. Работает **пока приложение открыто** (при старте и раз в час).

**Порядок шагов** (важен для качества архива):

1. **Синхронизация кафедр и преподавателей** — если справочник пуст или по расписанию.
2. **Проверка словаря тегов** (шаблоны по умолчанию, если словарь пуст).
3. **Пересчёт тегов постов** — если справочники только что заполнились, а в архиве уже есть посты.
4. **Догрузка из ВК** и **обновление метрик** — только после подготовки.

Без кафедр, преподавателей и активных тегов в словаре новые посты из ВК не получат привязку к авторам и хэштеги из вкладки «Тэги».

## Устранение неполадок

- **Видео не скачивается** — cookies в `.env`, FFmpeg в PATH для yt-dlp.
- **База заблокирована** — не открывайте `Archive.db` в сторонних редакторах во время работы приложения.
- **Ошибки** — см. «О программе» или `logs/errors.log`.

Проект разрабатывается в рамках дипломной работы ЧИБГУ.
