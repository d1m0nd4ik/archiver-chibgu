# VK Archiver CHIBGU

Десктопное приложение для скачивания, обработки, индексации и анализа контента из сообществ ВКонтакте. Разработано для ведения архива публикаций с привязкой к преподавателям и кафедрам ЧИБГУ.

Интерфейс построен на PySide6 (Qt). Данные хранятся в SQLite (`Archive.db`), медиафайлы — в каталоге `Exports_data` с разбивкой по дате публикации.

---

## Основные возможности

### Загрузка и сохранение медиа

- Скачивание постов из группы ВК по токену API (фото, видео, клипы).
- Выбор периода: час, день, неделя, месяц, год, весь архив или произвольный диапазон дат.
- Фотографии сохраняются в **JPEG** в исходном виде, без сжатия и без конвертации в другие форматы.
- Видео загружается **без перекодирования** (прямая ссылка или yt-dlp); приложение не выполняет оптимизацию и не меняет кодек файлов.
- Превью к роликам сохраняются отдельными файлами `.jpg`.
- Поддержка ограниченного контента через cookies для yt-dlp (файл `cookies.txt` или настройки в `.env`).

### Индексация и поиск

- Сохранение текста постов, тегов и метрик (лайки, комментарии, репосты) в базе данных.
- Полнотекстовый поиск по архиву (SQLite FTS5) на странице «Поиск в архиве».
- Автоматическая генерация хештегов из текста (Natasha / pymorphy3).

### Хранилище и статистика

- Просмотр загруженных постов в виде карточек с медиа, встроенным видеоплеером и подгрузкой порциями.
- Обновление статистики постов напрямую из VK без повторной загрузки файлов.
- Аналитика по периодам: топ постов по лайкам, комментариям, репостам и сводной популярности.
- Экспорт отчётов в CSV и Excel (таблицы и диаграммы).

### Преподаватели и кафедры

- Справочник кафедр и преподавателей с уникальными хештегами (`#Фамилия_И_О`, `#Кафедра_...`).
- Синхронизация списков с сайта университета: парсинг страниц кафедр, загрузка состава, отдельно — преподаватели колледжа.
- Ручное добавление, редактирование и удаление записей в интерфейсе «Кафедры и преподаватели».
- Привязка постов к автору (преподавателю) и кафедре в базе данных.
- Страница «Преподаватели в постах»: сводка активности за период и экспорт публикаций в Word (`.docx`).

### Оформление и настройки

- Три режима темы: светлая, тёмная и «как в системе».
- Единый стиль панелей, таблиц и кнопок с корректным отображением при смене темы.
- Сохранение токена VK, ссылки на сообщество и темы в файле `.env`.
- Проверка токена и группы из настроек, создание ярлыка на рабочем столе (Windows).

## Требования

- Python 3.9 или новее
- Доступ в интернет для VK API и синхронизации кафедр
- Токен доступа VK API ([vkhost.github.io](https://vkhost.github.io/))

## Установка

### 1. Клонирование и виртуальное окружение

```bash
git clone <URL-репозитория>
cd Archiver-CHIBGU
python -m venv .venv
```

**Windows (PowerShell):**

```powershell
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

**Linux / macOS:**

```bash
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Конфигурация

Скопируйте пример и заполните значения:

```bash
copy .env.example .env
```

Параметры в `.env`:

| Переменная       | Описание                                                                                 |
| -------------------------- | ------------------------------------------------------------------------------------------------ |
| `VK_TOKEN`               | Токен VK API                                                                                |
| `VK_GROUP_LINK`          | Ссылка, короткое имя или числовой ID сообщества            |
| `POST_COUNT`             | Сколько постов запрашивать за раз (по умолчанию 20)      |
| `APP_THEME`              | `light`, `dark` или `system`                                                            |
| `VK_COOKIES_FILE`        | Путь к cookies.txt для yt-dlp (необязательно)                               |
| `VK_USE_BROWSER_COOKIES` | `1` — читать cookies из браузера (на Windows часто ненадёжно) |
| `VK_COOKIES_BROWSER`     | Список браузеров:`edge,chrome,firefox`                                          |

При первом запуске файл `.env` создаётся автоматически, если его нет.

## Стек технологий

- **GUI:** PySide6
- **VK:** vk_api, requests
- **Медиа:** Pillow (метаданные изображений), yt-dlp (загрузка видео)
- **NLP:** natasha, pymorphy3
- **Парсинг сайта кафедр:** BeautifulSoup4, requests
- **Отчёты:** openpyxl, python-docx
- **Конфигурация:** python-dotenv

## Устранение неполадок

- **Видео не скачивается** — укажите `VK_COOKIES_FILE` в `.env` или проверьте доступность ролика в браузере; для части ссылок yt-dlp может потребовать установленный FFmpeg в системе, но приложение сами файлы не перекодирует.
- **Ошибка токена** — получите новый токен с правами на группу и сохраните в настройках.
- **База заблокирована** — не открывайте `Archive.db` в стороннем редакторе во время работы приложения; используется режим WAL.
- **Экспорт Word не работает** — установите зависимость: `pip install python-docx`.
