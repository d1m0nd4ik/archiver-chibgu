# 🎬 VK Archiver CHIBGU

Приложение для автоматического скачивания, оптимизации и индексации контента из ВКонтакте.

## ✨ Возможности

- 📥 **Загрузка контента** — фото и видео из групп ВКонтакте
- 🖼️ **Оптимизация фото** — конвертация в WebP (сжатие до 50%)
- 🎥 **Обработка видео** — конвертация в MP4 (H.264) с улучшением качества
- 🏷️ **Авто-тегирование** — генерация хештегов на основе анализа текста
- 🔍 **Быстрый поиск** — индексация в SQLite базе данных
- 🌙 **Темная тема** — современный интерфейс на PySide6

## 📋 Требования

- Python 3.9 или выше
- FFmpeg (обязательно!)

## 🛠 Установка FFmpeg

### Windows
1. Скачайте с [gyan.dev/ffmpeg](https://www.gyan.dev/ffmpeg/)
2. Распакуйте в `C:\ffmpeg`
3. Добавьте `C:\ffmpeg\bin` в переменную среды PATH
4. Проверьте: `ffmpeg -version` в командной строке

### macOS
```bash
brew install ffmpeg
